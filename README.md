**Wi-Fi roaming testing script**

Displays wireless statistics to screen, showing when Wi-Fi client roams

Saves output to CSV (rewtires file every time)

**Windows**

Install:
```
pip install -r requirements.txt
```

Run:
```
python roamwindows.py
```
